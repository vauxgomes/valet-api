# Valet

## Descrição
Uma API para detecção de anomalias em redes

## Instalação

### Ambiente virtual
```sh
# Criação
python3 -m venv venv/

# Ativação
source venv/bin/activate

# Requerimentos
(venv) % pip install -r requirements.txt

# List packages
(venv) % pip list

# Desativação
(venv) %  deactivate
```
